## Getting Started <a name = "getting_started"></a>

Copy the content in app/code/RG/Tiktok  Folder


### Installing

```
bin/magento module:enable RG_Tiktok --clear-static-content

bin/magento setup: upgrade

bin/magento s:d:c

bin/magento c:f

```
